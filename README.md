# cobalt2
Cobalt 2 Theme for Webstorm


Inspired by https://github.com/wesbos/cobalt2 but with my own touch.


## Install

Webstorm --> Preferences Editor --> Color Scheme --> Settings (Gear-Wheel) --> Import Scheme

and simply import the .icls file.
